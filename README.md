### What I Do

I am a 6th semester Software Engineering student at FET Neiva with a background in electronics and network support. I build full-stack web applications using React, Next.js, Node.js, and Laravel. My work focuses on creating practical solutions — from AI-powered chatbots to interactive mapping platforms.

My flagship project, **Al Son del Huila**, is a tourism platform that uses Groq/LLaMA AI with RAG to help travelers discover destinations in Huila, Colombia. It won a scholarship from the Gobernacion del Huila.
